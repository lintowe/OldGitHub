How it works

OldGitHub rewrites the GitHub web UI in your browser to match the 2012-2013 layout. It does this only on github.com tabs — no data leaves your machine, no token is required.

Some pages (such as Marketplace or Settings) keep the modern UI because there isn't a faithful 2013 equivalent.

You can also flip the theme from any GitHub page using the sun/moon icon in the header bar.
