How it works

OldGitHub rewrites the GitHub web UI in your browser to match the 2012-2013 layout. Authentication is your existing github.com session cookie.

A few pages are JS-only React forms that can't be re-rendered without their runtime (create-repo, create-issue, login, signup, account-settings sub-pages, gist.github.com). On those the themed top bar stays in place but GitHub's native body renders below it.

Flip the theme from any GitHub page using the sun/moon icon in the header bar.

Privacy

OldGitHub reads the current GitHub page, URL, username, and signed-in state to rebuild the interface in this browser. Page data is not logged or sent to the developer, advertisers, or any service outside GitHub.

Your theme choice is the only stored setting. Chrome may sync it to your other signed-in browsers when Chrome Sync is enabled.
